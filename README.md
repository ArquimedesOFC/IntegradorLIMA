# Multiverso TCG - Loja Digital de Cartas

O **Multiverso TCG** é uma plataforma de e-commerce focada em cartas colecionáveis dos jogos **Magic: The Gathering**, **Pokémon TCG** e **Yu-Gi-Oh!**. O projeto foi desenvolvido como parte do Projeto Integrador da disciplina de **LIMA** (Linguagem de Marcação) no **SENAI**.

### Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **Acessibilidade Web**

## Funcionalidades

O projeto simula uma loja de cartas com as seguintes funcionalidades principais:

- **Catálogo de Produtos**: Exibição de cartas colecionáveis em formato de grid.
- **Páginas de Produto**: Detalhes de cada carta, com imagem, descrição e preço.
- **Formulários de Login e Cadastro**: Sistema para criação de contas e login de usuários.
- **Carrinho de Compras e Checkout**: Funcionalidade para adicionar cartas ao carrinho e finalizar compras.
- **Página de Atendimento e Contato**: Formulário de contato com campos para nome, email e mensagem.
  
### Estrutura do Projeto

- **Home (index.html)**: Banner principal e destaques das cartas.
- **Produtos (produtos.html)**: Catálogo completo com cartas.
- **Páginas de Produto (produto1.html até produto20.html)**: Detalhes de cada carta.
- **Sobre (about.html)**: Página institucional apresentando a loja.
- **Atendimento (contact.html)**: Formulário de contato.
- **Login e Cadastro (login.html e register.html)**: Formulários para criação de conta e login.
- **Carrinho e Finalização (cart.html e checkout.html)**: Simulação de compra.
